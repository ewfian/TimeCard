## Getting Started
### Install Global Dependencies

```bash
$ npm install -g gulp
```

### Build & Development
```bash
# install local dependencies
$ npm install

# build package to ./dist
$ gulp

# start the dev server and browser to http://localhost:3000
$ gulp watch
```
